1.概述
本文档是散热器自动试验软件的使用说明文档，旨在为用户提供软件安装、配置、操作及维护的详细指导。文档内容涵盖软件运行环境、功能模块、操作流程等关键信息，帮助用户快速掌握软件使用方法，确保散热器测试工作的高效、准确执行。本文档适用于软件管理员、测试工程师及相关技术人员。

2.运行环境
2.1硬件条件
配置项	要求
处理器	Intel 12代酷睿 i5 及以上
内存	8GB 及以上
硬盘空间	至少 512GB 可用空间
接口	以太网接口（用于网络通信）
其他	稳定电源、鼠标、键盘、显示器（分辨率 1920×1080）
2.2软件条件
操作系统：Windows 10 专业版
数据库：SQL Server 2008 R2（需安装 Management Studio）
驱动及库文件：
LabVIEW VISA 库（版本 2018 及以上）
LabVIEW 运行引擎（LabVIEW Runtime Engine 2018 32位）
NXI-6110设备驱动



3.软件主要功能
本软件通过与设备网络中各仪器设备进行通信，实现对电源、风扇、供风系统、供水系统等的自动化控制，模拟特定测试环境，并实时采集电压、温度、电源功率等关键数据并记录存储，从而完成不同散热器在不同模拟工况下的温度变化特性测试。

4.软件安装与运行
4.1软件安装
1.将软件安装包解压至指定目录。
2.运行安装程序Setup.exe，按照向导提示完成安装（为避免系统盘空间不足，建议您将软件安装至D盘）。

3.安装完成后，配置数据库连接参数（服务器地址、数据库名称、用户名及密码）。
4.重启计算机，确保所有驱动及服务正常加载。
4.2启动前准备
1.确保所有测试仪器已正确连接到计算机，并开机通电。
2.启动 SQL Server 服务（安装后默认开机自启动，无需再次手动启动）。
3.启动软件：
双击桌面快捷方式"传感器自动测试软件"，或导航至软件安装目录（默认路径：C:\Program Files (x86)\散热器试验软件.exe），双击主程序散热器试验软件.exe。
软件启动后，将自动连接数据库。
连接成功后，将进入软件登陆界面。
4.3运行注意事项
软件运行期间，请勿关闭 SQL Server 服务或断开仪器连接。
软件运行期间，请勿强制关闭程序，以防数据丢失或设备异常。
测试过程中，避免操作其他占用系统资源较大的程序，以防影响测试稳定性。

5.功能模块
软件架构实行模块化设计及管理，各模块主要配置信息会在退出软件时自动保存，下次打开时将加载上次关闭时的参数，参数与用户名进行绑定，登录时会加载对应的配置信息。
5.1主模块
系统主界面，提供各功能模块的入口导航及当前测试场景的模块显示。
界面说明（见下图）：

下方四个区块：为当前【场景】下所使用的四个功能模块，可通过场景配置模块进行配置。
上方区域：功能导航、场景选择及主要操作组件。
点击【配置】进入配置选择页；
点击【场景】进入场景选择页；
右上方依次为：启用温度及电压监控按钮、开始测试按钮、关闭加热电源按钮；
红色电源键为退出软件按钮。
上方菜单栏：提供切换用户登录及用户管理界面入口。


5.2用户管理模块
软件提供用户及权限管理功能，可通过主界面的菜单栏【用户管理】进入管理界面。

支持以下操作：
添加用户
搜索用户
删除用户
编辑用户
编辑权限
说明：
用户信息保存在数据库的KD0028库中。
软件安装后会生成一个用户名为Admin的系统管理员账号，默认密码为admin，该账号默认无法删除。
用户权限分为多级权限，仅一级权限（系统管理员）可执行系统配置及用户管理操作。

5.3场景配置模块
场景配置通过主界面上方【配置】选择页进入。

功能说明：
左侧为当前的场景模块列表，软件初始内置了四个默认场景，这四个场景无法修改模块选择或删除，仅支持修改场景名称。
用户可通过右侧操作按钮进行增加场景、删除场景等操作。
新增的自定义场景可在界面中自定义模块选择，可选模块包括：
万瑞达加热模块
TDK 加热模块
温度采集模块
电压采集模块
风扇控制模块
供风系统模块
供水系统模块
数据显示模块
用户可自定义选择组合四种模块并命名进行测试。
场景配置完成后点击右侧【保存配置】按钮即可应用到软件中；退出配置界面后主界面场景选择页将自动刷新，加载最新的场景配置。

5.4仪器配置模块
仪器配置模块通过主界面的配置选择页点击【设备配置】按钮进入。

支持配置的仪器：
万瑞达电源
TDK 电源
N3411E 小电源（NGI 供电电源）
GM10（热电偶采集）
M203（NTC 采集）
NGI 电压采集卡
小风扇电源 N36209
操作说明：
各个仪器可通过对应界面左侧的复制按钮将设备或通道的参数复制到其他同类型的设备或通道。
对于 NGI 电压采集卡、GM10 与 M203 三种仪器配置，左侧除复制按钮外还提供通道激活勾选框，勾选后将在对应模块中展示；复制通道数据后将默认勾选对应通道。

5.5数据管理模块
数据管理模块通过主界面的配置选择页点击【数据配置】按钮进入。

该模块主要用于对系统全部数据的管理及保存相关参数的修改。

5.6万瑞达加热模块
此模块用于十个万瑞达电源的参数便捷设置及关键数据查看，上方菜单栏可切换至波形显示。

功能说明：
主界面可切换电源模式：
恒功率模式：电压上限设定值将固定为200V，无法通过此界面修改，可通过修改设定功率的方式自动修改电流至目标值。
常规模式：手动设置电压、电流参数。
参数编辑完成后，通过右侧选中框勾选需要应用的电源，点击【应用按钮】，参数将设定至指定电源，列表中对应电源将以淡蓝色高亮显示。
随后点击左上【启动电源按钮】，即可启动列表框高亮显示的电源。
启动电源成功后，所选电源对应的相关数据将自动匹配至【数据管理模块】并勾选，波形显示界面也会同步更新。

5.7TDK 加热模块
该模块控制仪器包含四个 TDK 加热电源及两台功率板低压供电电源 N3411E。

功能说明：
TDK电源模式同样包含恒功率模式及常规模式，电压默认设置10V且不可在瓷介民修改。操作方法与【万瑞达加热模块】一致，模块同样支持 TDK 电源的电压、电流、功率的波形同步显示。请参考 5.6。
电源 N3411E 界面：可通过模块上方的选择页点击【供电电源NGI】进入，每台电源包含两个可用通道，设置好电压及电流参数后点击左侧蓝色电源键启动电源；右侧为当前电压及电流的实时显示。


5.8温度采集模块
温度采集模块用于与 GM10（热电偶）、M203（NTC）通讯并显示实时数据。

说明：
只有在【设备配置模块】中勾选激活后，才会在该界面展示。
波形显示界面也会同步更新并显示。
GM10：包含 50 个通道。
M203：每个设备 8 个通道，配有两台，共 16 个通道。

5.9电压采集模块
电压采集模块用于与 NGI 采集卡 NXI-6110 通讯并显示实时数据。

说明：
只有在【设备配置模块】中勾选激活后，才会在该界面展示。
波形显示界面也会同步更新并显示。
每张采集卡包含 8 个通道，配有四张同型号采集卡，共 32 个通道。

5.10风扇控制模块
该模块用于控制电源 N36309 向 PWM 调速模块供电，并与调速模块进行通讯进而控制风扇转速。

操作说明：
设置好电压及电流参数后点击【启动电源】，电源便会开启供电并发送配置中加载的默认转速（占空比）。

5.11供风系统模块
该模块与供风系统的 PLC 进行通讯，控制设备参数的修改，并采集 PLC 数据域内的传感器及内部参数等数据显示在模块界面。


供风系统支持三种模式：
模式	说明
手动频率	手动设定风机频率
自动风量	自动控制目标风量
平均风速	按照平均风速控制
修改模式并设定好目标数值后，点击【启动供风】即可依据设定参数控制风机的启停。

5.12供水系统模块
该模块与供水系统的 PLC 进行通讯，控制设备参数的修改，并采集 PLC 数据域内的传感器及内部参数等数据显示在模块界面。

供水系统包含供水及控温两种功能：
供水功能
供水分为通道一与通道二：
通道一：高流量通道（0～80 L/min）
通道二：低流量通道（0～16 L/min）
可通过模块上方的通道选择下拉框进行切换，并通过【流量设定】修改目标流量。
供水前可进行气密操作来检查通道的气密性，点击【启动气密】按钮对选中通道进行气密检查。
点击【启动供水】按钮对泵执行启停操作。
停止供水后可点击【启动吹水】对管路进行排水操作。
控温功能
可通过上方的【启动内循环】进行启停操作。
修改【温度设定】值进行温度设定操作。
当设备报警后，排除故障点后可通过点击【报警复位】来复位报警状态。

5.13数据显示模块
数据显示模块用于对在【数据配置模块】中所有选中的参数进行展示，并对温度进行直方图展示。

功能说明：
直方图展示仅会展示选中的温度通道（包括 GM10 及 M203 的采集数据）。
可通过上方的选择页点击进入直方图界面。
每个直方图界面最多展示 22 个通道的数值。
当选中的通道数大于 22 时，会自动新增一个选择页进行展示，最多三个直方图选择页，共 66 个通道。
点击【查看数据】按钮可对当前测试的数据进行分析并保存对应数据。

6.软件操作流程说明
用户登录 → 确认自检状态 → 场景配置 → 设备配置 → 数据配置 → 测试执行 → 数据处理及分析`
6.1用户登录
启动软件后会自动弹出用户登录界面。

输入正确的用户名、工号及密码后，软件将自动加载对应用户名的配置。配置保存在安装目录下的Cfg文件夹中，以不同的用户名命名为文件夹名。

6.2确认自检状态
登录成功后，确认各仪器设备自检状态正常，确保所有设备处于可用状态后再进行后续操作。
6.3场景配置
根据测试需求，在场景配置模块中选择或新建合适的测试场景，配置所需的功能模块组合。
6.4设备配置
进入仪器配置模块，对各测试仪器进行参数配置，并激活温度和电压采集需要使用的通道。
6.5数据配置
进入数据管理模块，对需要采集和保存的数据参数进行选择和配置。
6.6测试执行
配置完成后，在主界面点击【开始测试】按钮启动测试流程，系统将按照配置的参数自动执行测试并实时采集数据，经过【数据配置】中的测试时长之后，完成测试，自动停止记录数据并弹出数据分析界面。
6.7数据处理及分析
测试完成后，通过数据显示模块的【查看数据】按钮对测试数据进行分析，并保存测试结果。


7.常见问题与处理
问题描述	可能原因	处理方法
软件无法连接数据库	SQL Server 服务未启动或连接参数错误	检查 SQL Server 服务状态，核实数据库连接参数
仪器无法通信	网络连接异常或仪器 IP 配置错误	检查网络连接，核实仪器 IP 地址配置
数据采集异常	仪器通道未激活或驱动未正确安装	在设备配置模块中确认通道激活状态，重新安装驱动
软件启动失败	运行环境不满足要求	检查操作系统版本及 LabVIEW 运行引擎是否正确安装
设备报警	系统检测到故障点	排除故障后点击【报警复位】恢复正常状态
