- 每个 HTML 页面通过 `<body data-page="...">` 声明页面类型，`script.js` 据此进行路由式分支初始化。
- 所有可复用的 DOM 片段（头部、尾部）以独立 IIFE 脚本 (`header.js`/`footer.js`) 提供，通过 `<template>` 生成节点后替换或插入 `.page-shell` 内的占位元素。
- 数据文件统一以 `<script src="data/xxx.js?v=timestamp"></script>` 形式动态加载，并以 `window.__BLOG_*__` 全局变量暴露，由 `loadDataScript` 配合校验函数保证可用性。
- 向用户可见的字符串一律经 `escapeHtml` 转义后再拼入 innerHTML，防止 XSS。
- URL 段与分类/标签等用户可控输入先经 `sanitizeSegment` 过滤非安全字符后再参与路径拼接。
- CSS 类名遵循 BEM 风格前缀（如 `post-card`、`archive-card`、`fragment-entry`、`site-header`），并通过 `is-active`/`is-open` 等修饰类表达状态。
