- 每个分类对应 `posts/<folder>/` 下的子目录，且 `data/categories.js` 中的 `folder` 字段与之一一对应，作为路由与归档的约定键。
- 文章元数据对象固定包含 `id`、`title`、`excerpt`、`summary`、`description`、`category`、`categoryOrder`、`folder`、`date`、`dateLabel`、`tags`、`readingTime`、`readingMinutes`、`wordCount`、`wordCountValue`、`featured`、`pinned`、`cover`、`path`、`sourcePath`、`sourceDir`、`imageDir`、`showInRecent`、`recentOrder`、`showInArchive`、`archiveOrder` 等字段，保持跨条目一致。
- 单篇文章的 `.js` 与 `.json` 双份产物必须完全等价，仅前者通过 `window.__BLOG_*` 暴露给页面，后者用于 API 或离线消费。
- 碎片条目使用 `YYYY-MM-DD-HH-mm-ss` 形式的 `id`，并以 ISO 8601 时间戳存入 `date` 字段，同时保留可读的 `timeLabel`。
